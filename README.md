# 🛍️ Customer Segmentation App

## 📌 Project Overview

This project performs customer segmentation using K-Means clustering on mall customer data.

The goal is to identify different customer groups based on:

* Annual Income
* Spending Score

---

## 🚀 Features

* Upload dataset
* Perform EDA (visualizations & stats)
* Apply K-Means clustering
* Visualize customer segments
* Generate business insights

---

## 📊 Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib
* Scikit-learn

---

## 📈 Key Insights

* Identified VIP customers (high income + high spending)
* Found target customers (high income + low spending)
* Showed that income does not always correlate with spending

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
